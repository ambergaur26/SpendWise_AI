SpendWise AI: Agentic Expense Manager
SpendWise AI is an intelligent expense tracking chatbot that leverages FastAPI, Groq Llama3, and PostgreSQL to transform natural language into structured financial data. It features an automated pipeline for database insertion via JSON, offering real-time balance tracking, 7-day spending insights, and proactive 80% overspending alerts. Built with a Streamlit interface, it ensures reliable financial management through robust validation and error handling.

 Key Features
Natural Language Processing: Seamlessly converts conversational text into structured JSON for automated expense logging.

AI-Powered Insights: Utilizes Groq Llama3 to categorize spending and provide intelligent 7-day financial summaries.

Real-time Budget Alerts: Automatic notification system that triggers when spending reaches 80% of the set budget.

Live Balance Tracking: Maintains a persistent PostgreSQL database to provide up-to-the-minute balance and history updates.

Robust Error Handling: Integrated validation layers to ensure AI outputs are accurate and database-ready.

Full-Stack Architecture: A high-performance stack combining a FastAPI backend, Streamlit frontend, and SQL-driven logic.
